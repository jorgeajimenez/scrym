# User Persona: Offensive Coordinator (OC)

**Role:** The Play Caller.
**Environment:** Rapid cadence (40-second play clock). Needs flow and rhythm.
**UX Requirement:** **"The Menu"** - Large touch targets. One-tap actions. No scrolling.

## Core Use Cases

### 1. Instant Play Selection
**Goal:** Select the next play before the play clock hits 25.
- **Workflow:**
    1.  Previous play ends. System auto-updates down/distance.
    2.  **UI:** Displays 3 massive cards: **"PASS (Safe)"**, **"RUN (Inside)"**, **"SHOT (Deep)"**.
    3.  **Action:** OC taps "PASS".

### 2. "Send to QB" (One-Tap Comms)
**Goal:** Relay the play to the quarterback/sideline signal caller instantly.
- **Workflow:**
    1.  OC taps the selected play card.
    2.  **UI:** Screen flashes the **"WRISTBAND CODE"** (e.g., **"BLUE 42"**) in full-screen bold text.
    3.  **Action:** Sideline assistant signals "Blue 42" to the QB.
    4.  **Feedback:** Button turns Green ("Sent").

### 3. Personnel Grouping Match
**Goal:** Exploit defensive fatigue or substitution patterns.
- **Workflow:**
    1.  Entering the Red Zone.
    2.  **UI:** A simple alert icon appears: **"USE HEAVY (13)"**.
    3.  **Action:** OC yells "13 Personnel!" to the sideline sub coordinator.
